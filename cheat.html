VS Code的快捷鍵 - Windows 版本

導航

  • Ctrl + B：切換檔案總管
  • Ctrl + P：按名稱搜尋檔案
  • Ctrl + Shift + P：快速進入任何設置
  • Ctrl + `：切換終端機開關

編輯

  • Ctrl +/-：放大/縮小
  • Ctrl + Shift + F：在整個專案中搜尋單詞或短語
  • Ctrl + W：關閉當前檔案
  • Ctrl + Shift + T：重新打開最近關閉的檔案
  • Ctrl + D：選擇當前單詞的下一個出現位置以進行多重編輯
  • Alt + 上/下：移動當前行
  • Alt + Shift + 上/下：複製當前行
  • Ctrl + X：剪切當前行
  • Ctrl + 左/右：移動到行的開始/結束
  • Alt + 左/右：向左/右移動一個單詞

Finder/Explorer 整合

  • 右鍵點擊文件夾：選擇“在文件夾中查找”以僅在特定目錄中搜尋
  • 右鍵點擊檔案：選擇“在文件資源管理器中顯示”以打開檔案位置
